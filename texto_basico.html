Histórico:

A linguagem de programação Swift foi desenvolvida pela Apple em 2014, com a intenção de ser uma linguagem moderna e segura para o desenvolvimento de aplicativos para iOS, macOS, watchOS e tvOS. Ela substituiu a antiga linguagem Objective-C como a principal opção para o desenvolvimento de aplicativos para essas plataformas.

Características:

- Sintaxe simples e expressiva: A sintaxe do Swift foi projetada para ser concisa e legível, tornando o código mais fácil de escrever e entender.


- Segurança e confiabilidade: O Swift possui recursos de segurança que ajudam a evitar erros comuns de programação, como acesso a variáveis não inicializadas ou ação indesejada em memória protegida.


- Inferência de tipos: O compilador do Swift é capaz de inferir o tipo de uma variável com base no valor atribuído a ela, evitando a necessidade de especificar o tipo explicitamente.


- Tratamento de erros: O Swift suporta um modelo de tratamento de erros que permite lidar com falhas e recuperação em tempo de execução de forma estruturada.


- Performance: O Swift foi projetado para ter um desempenho rápido e eficiente, tornando-o adequado para o desenvolvimento de aplicativos que requerem resposta rápida.


instalação:

- Para desenvolver em Swift, é necessário ter o Xcode, a IDE oficial da Apple, instalado no macOS. O Xcode já inclui a versão mais recente do compilador do Swift, bem como outras ferramentas necessárias para o desenvolvimento iOS, macOS, watchOS e tvOS.



Variáveis

- No Swift, as variáveis são declaradas usando a palavra-chave "var" e os valores podem ser alterados uma vez que a variável tenha sido inicializada. Exemplo: var nome: String = "Felipe Miranda"


Operadores

- O Swift suporta os operadores aritméticos (+, -, *, /), operadores de comparação (>, <, ==, !=), operadores lógicos (&&, ||, !) e muitos outros operadores comuns encontrados em outras linguagens de programação.


Tipos de Dados:

- O Swift possui tipos de dados básicos, como Int, Double, String e Bool. Além disso, o Swift também suporta tipos opcionais, que podem representar um valor nulo ou um valor existente.


Funções:

- No Swift, as funções são declaradas usando a palavra-chave "func" e podem ter zero ou mais parâmetros de entrada. As funções também podem ter um tipo de retorno opcional. Exemplo: func soma(a: Int, b: Int) -> Int { return a + b }


Arrays:

- No Swift, os arrays são coleções ordenadas de elementos do mesmo tipo. Eles são declarados usando colchetes e podem ser mutáveis (var) ou imutáveis (let). Exemplo: var numeros: [Int] = [1, 2, 3, 4, 5]


Essas são apenas algumas das características da linguagem de programação Swift. A linguagem continua a evoluir e expandir suas funcionalidades, tornando-a cada vez mais popular para o desenvolvimento de aplicativos para dispositivos Apple.


Exemplos básicos

1) Hello World:


```
import Swift
print("Hello, World!")
```
Este é o exemplo básico de programação, onde a mensagem "Hello, World!" é exibida no console.



2) Soma:


```
import Swift

let num1 = 5
let num2 = 10
let result = num1 + num2

print(result)
```
Neste exemplo, duas variáveis `num1` e `num2` são inicializadas com os valores 5 e 10, respectivamente. A soma dos números é armazenada na variável `result` e é impressa no console.



3) Estrutura de decisão:


```
import Swift

let num = 7

if num % 2 == 0 {
print("O número é par")
} else {
print("O número é ímpar")
}
```
Neste exemplo, é utilizada uma estrutura de decisão para verificar se o número armazenado na variável `num` é par ou ímpar. Se o resto da divisão de `num` por 2 for igual a 0, a mensagem "O número é par" é exibida; caso contrário, a mensagem "O número é ímpar" é exibida.
