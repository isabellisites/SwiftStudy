Ponteiros:

Diferente de algumas outras linguagens, a linguagem Swift não possui ponteiros explícitos. Isso significa que não há a necessidade de lidar diretamente com ponteiros, como em C ou C++. No entanto, a linguagem Swift possui um tipo chamado "UnsafePointer" que permite trabalhar com ponteiros e acessar memória de forma não segura quando necessário.



Structs:

A linguagem Swift encoraja o uso de structs (estruturas) para definir tipos de dados complexos. As structs em Swift são usadas amplamente para criar tipos de dados como inteiros, strings, vetores, etc. Elas são muito eficientes em Swift devido ao seu comportamento de cópia e tamanho fixo. Além disso, as structs em Swift suportam funcionalidades avançadas, como métodos, propriedades e até mesmo protocolos.

Arquivos:

A linguagem Swift possui uma sintaxe fácil e legível para manipulação de arquivos. Ela fornece várias APIs para ler e escrever em arquivos. Para ler um arquivo, você pode usar a classe "FileManager" para localizar o arquivo e depois usar a classe "FileHandle" para manipular os dados contidos no arquivo. Para escrever em um arquivo, você também pode usar a classe "FileHandle" para abrir o arquivo em modo de escrita e gravar os dados desejados.

Exemplos

1. Exemplo de uso de ponteiros:


```swift
func increment(_ value: UnsafeMutablePointer) {
value.pointee += 1
}

var number = 10
increment(&number)
print(number) // Output: 11
```



2. Exemplo de uso de structs:


```swift
struct Point {
var x: Int
var y: Int

mutating func moveBy(deltaX: Int, deltaY: Int) {
x += deltaX
y += deltaY
}
}

var point = Point(x: 0, y: 0)
point.moveBy(deltaX: 5, deltaY: 3)
print(point) // Output: Point(x: 5, y: 3)
```



3. Exemplo de leitura e escrita de um arquivo:


```swift
guard let path = Bundle.main.path(forResource: "file.txt", ofType: nil) else {
fatalError("File not found")
}

let fileURL = URL(fileURLWithPath: path)
let fileHandle = try FileHandle(forReadingFrom: fileURL)

let data = fileHandle.readDataToEndOfFile()
let content = String(data: data, encoding: .utf8)

print(content ?? "") // Output: Contents of the file

fileHandle.closeFile()
```
